Twitter Post - Settings Guide



  1. Choose a background image or leave blank.
  2. Enter your Discord Webhook URL (How to Create a Discord Webhook).
  3. Enter the Name of Posting User.
  4. This can be anything you want.
  5. Enter the Twitter Username.
  6. This must be a real Twitter Username.
  7. Do not include the "@" Symbol.
  8. Enter the URL of Twitter Post.
  9. This is the Twitter Post you would like to post to Discord.
  10. Choose Text to Speech Setting.
  11. If this is set to "On", Discord will read your message aloud using TTS.
